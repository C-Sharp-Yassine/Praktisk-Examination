# Praktisk-Examination
ChatApp med Socket.IO

## Hur man kör programmet ChattApp

1. Öppna projektet i Visual studio eller JetBrains.
2. Kontrollera att paketet SocketIOClient är installerat.
3. När programmet start:
   - skriv in ditt namn när du får frågan "Ange ditt namn:"
   - Vänta tills du ser "Connected"
   - Börja skriva meddelanden i chatten.
     
## Kommando i chatten:
   - tex: skickar meddelande till chatten
   - /quit: avslutar programmet och kopplar från servern.
